# React To Do App

A sleek, modern Todo List web application built with React.js and Vite. Featuring a stunning colorful dark mode with neon/cyberpunk-inspired aesthetics — vibrant glows, glassmorphism effects, and smooth animations for a futuristic feel.

Live demo: [https://vinimathew22.github.io/react-todo/](https://vinimathew22.github.io/react-todo/)

## Features

Add new tasks effortlessly
Mark tasks as complete with checkmarks
Delete tasks
Responsive and mobile-friendly design

## Technologies used

React – Component-based UI
Vite – Blazing-fast development server and builds
Pure CSS – For the custom neon dark mode styling (no external libraries needed)
